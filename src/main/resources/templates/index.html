Giới thiệu bản thân

Mình có niềm đam mê mạnh mẽ với phát triển phần mềm, đặc biệt là xây dựng các ứng dụng web có giao diện trực quan và luồng xử lý dữ liệu rõ ràng. Trong quá trình học tập, mình chủ động tham gia các đồ án nhóm, nơi mình rèn luyện kỹ năng giao tiếp kỹ thuật, quản lý tiến độ và xử lý vấn đề khi dự án gặp áp lực thời gian.

Ngoài kiến thức trên lớp, mình thường tự học qua tài liệu chính thức, thực hành mini project và đọc mã nguồn mở để cải thiện tư duy lập trình. Mục tiêu của mình không chỉ là viết code chạy được, mà còn viết code sạch, có kiểm thử và dễ bảo trì trong dài hạn.

Java Spring Boot HTML/CSS JavaScript MySQL Git & GitHub

Hành trình phát triển

  • Hoàn thành các môn nền tảng về cấu trúc dữ liệu, giải thuật và lập trình hướng đối tượng, tạo nền tảng tốt cho việc xây dựng ứng dụng thực tế.
  • Thực hiện nhiều bài tập lớn sử dụng Java và Spring Boot, tập trung vào thiết kế API, kết nối cơ sở dữ liệu và tổ chức code theo mô hình chuẩn.
  • Cải thiện kỹ năng front-end để tạo giao diện rõ ràng, responsive và tối ưu trải nghiệm trên cả desktop lẫn mobile.

Mục tiêu gần nhất

Trong 12 tháng tới, mình muốn hoàn thiện một portfolio cá nhân với các dự án thực chiến, đặc biệt là ứng dụng quản lý sinh viên và hệ thống đặt lịch có phân quyền người dùng.

"Mỗi dòng code đúng hôm nay sẽ tiết kiệm rất nhiều giờ sửa lỗi cho ngày mai."
